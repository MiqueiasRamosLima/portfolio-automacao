// fluxo make.com — módulos em sequência

01
Receber briefing
Webhook aguarda POST com tipo, assunto, marca, tom e informações extras. Pode vir de formulário, Google Sheets ou CRM.
Webhooks › Custom Webhook
02
Montar prompt dinâmico
Variáveis do webhook são injetadas no prompt. O tipo de conteúdo define o system prompt e as instruções de formatação.
Tools › Set Variable
03
Gerar com GPT-4o
OpenAI recebe o prompt e retorna JSON com conteúdo, hashtags sugeridas e score de qualidade estimado.
OpenAI › Create a Completion
04
Salvar histórico no Supabase
Conteúdo gerado, prompt usado, tipo e timestamp são salvos. Permite auditoria, reaproveitamento e análise de performance.
Supabase › Insert a Row
05
Entregar no canal
Roteamento por tipo: LinkedIn → adiciona ao Google Sheets de aprovação; E-mail → envia via SMTP; Produto → atualiza registro no CRM.
Router + HTTP / Gmail / Sheets
