// Prompt enviado para GPT-4o via Make.com HTTP Module

SYSTEM:
Você é um qualificador de leads B2B especialista.
Analise o lead e retorne APENAS JSON válido com esta estrutura:

{
  "score": 0-100,             // probabilidade de conversão
  "classe": "hot|warm|cold",
  "potencial": "string",       // ticket estimado
  "prioridade": "string",     // urgência de contato
  "proxima_acao": "string",  // recomendação para SDR
  "analise": "string"         // resumo em 2-3 frases
}

USER:
Nome: {{lead.nome}}
Empresa: {{lead.empresa}}
Cargo: {{lead.cargo}}
Porte: {{lead.porte}}
Mensagem: {{lead.mensagem}}
-- Tabela criada no Supabase para armazenar leads qualificados

CREATE TABLE leads_qualificados (
  id          UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  created_at  TIMESTAMPTZ DEFAULT now(),

  -- Dados originais do lead
  nome        TEXT NOT NULL,
  empresa     TEXT,
  cargo       TEXT,
  porte       TEXT,
  mensagem    TEXT,

  -- Resultado da qualificação por IA
  score       INT CHECK(score BETWEEN 0 AND 100),
  classe      TEXT CHECK(classe IN ('hot', 'warm', 'cold')),
  potencial   TEXT,
  prioridade  TEXT,
  proxima_acao TEXT,
  analise     TEXT,

  -- Controle de CRM
  crm_id      TEXT,             -- ID retornado pelo Kommo
  status      TEXT DEFAULT 'novo'
);

-- Índice para busca rápida por classe
CREATE INDEX idx_leads_classe ON leads_qualificados(classe);
CREATE INDEX idx_leads_score  ON leads_qualificados(score DESC);

-- View para dashboard: top leads quentes
CREATE VIEW v_leads_hot AS
  SELECT nome, empresa, score, proxima_acao, created_at
  FROM leads_qualificados
  WHERE classe = 'hot'
  ORDER BY score DESC;
// Cenário Make.com — 6 módulos em sequência

1. Webhooks › Custom Webhook
   → Recebe POST com dados do lead
   → Extrai: nome, empresa, cargo, porte, mensagem

2. Tools › Set Variable
   → Monta o prompt com os dados do lead
   → Formata JSON de entrada para a IA

3. OpenAI (ChatGPT) › Create a Completion
   model: "gpt-4o"
   response_format: { type: "json_object" }
   → Retorna score, classe, análise, próxima ação

4. JSON › Parse JSON
   → Converte string da IA em variáveis utilizáveis
   → Mapeia score, classe, analise, proxima_acao

5. Supabase › Insert a Row
   table: "leads_qualificados"
   → Salva todos os dados + resultado da IA
   → Retorna UUID gerado para rastreio

6. HTTP › Make a Request (Kommo API)
   method: POST
   url: "https://sua-conta.kommo.com/api/v4/leads"
   → Cria lead com tags: hot/warm/cold
   → Atribui para SDR responsável automaticamente

// Tratamento de erros (módulo adicional)
7. Tools › Error Handler
   → Salva falhas no Supabase (tabela: logs_erros)
   → Envia alerta no Slack se score > 80 falhar
